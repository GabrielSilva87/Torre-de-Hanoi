# Torre de Hanoi

## Descrição

O projeto "Torre de Hanoi" é uma implementação do clássico jogo de quebra-cabeça. O objetivo do jogo é mover uma pilha de discos de uma estaca para outra, seguindo algumas regras simples.

## Regras do Jogo

- Apenas um disco pode ser movido por vez.
- Cada movimento consiste em pegar o disco do topo de uma das pilhas e colocá-lo em outra pilha.
- Um disco só pode ser colocado em uma pilha vazia ou em cima de um disco maior.

## Tecnologias Utilizadas

- JavaScript
- TypeScript
- HTML
- CSS
